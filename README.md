# CMS-Gut-Microbiome
Data analysis of Chinese Mystery Snail gut microbiomes. Associated with Alaei et al., in review. 

Raw sequencing data can be found at NCBI Repository Numbers PRJNA1310655 and PRJNA1436030 and were analyzed using QIIME2 v.2024.5.

**Venn-Diagram.R**
Counts shared ASVs between specified groups (e.g. environmental compartments) for construction of a Venn diagram using ggVennDiagram.

**PCoA.R**
Generates a PCoA of Bray-Curtis centriods for beta dispersion visualization.
